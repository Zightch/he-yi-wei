# 何意味

## 项目简介

这是一个基于拼音组合生成汉字排列的Python工具。项目从所有拼音为"he"、"yi"、"wei"的汉字中进行全排列组合，生成所有可能的三个汉字组合。

## 功能特点

- **多线程处理**：利用Python的threading模块实现并行计算，显著提高生成速度
- **进度显示**：在处理过程中显示实时进度条（单线程版本）
- **拼音数据解析**：支持解析标准拼音数据格式，处理多音字情况
- **智能去声调**：将带声调的拼音转换为无声调形式便于匹配

## 数据来源

项目使用来自[mozillazg/pinyin-data](https://github.com/mozillazg/pinyin-data)项目的`pinyin.txt`文件作为拼音数据源，该文件包含了Unicode汉字到拼音的映射关系。

## 使用方法

### 准备数据
1. 从GitHub下载拼音数据文件：
   ```
   https://github.com/mozillazg/pinyin-data/blob/master/pinyin.txt
   ```
2. 将下载的`pinyin.txt`文件放在项目根目录

### 运行程序
直接运行`main.py`文件：
```bash
python main.py
```

程序将：
1. 加载并解析拼音数据
2. 提取所有拼音为"he"、"yi"、"wei"的汉字
3. 生成所有可能的排列组合
4. 将结果保存到`何意味.txt`文件

## 输出格式

输出文件`何意味.txt`中，每行包含多个三个汉字的组合，以空格分隔，按照特定的格式组织：
- 每个"he"对应的汉字会开启一个新的主要分组
- 在每个主要分组内，按照"yi"对应的汉字进行分行

## 技术实现

### 核心功能
1. **拼音字典加载**：解析标准拼音数据格式，构建拼音到汉字的反向映射
2. **声调处理**：去除拼音中的声调标记，统一拼音格式
3. **组合生成**：计算所有可能的汉字组合（"he"×"yi"×"wei"）
4. **并行处理**：将计算任务分配到多个线程同时执行

### 性能优化
- 多线程并行计算
- 分批处理避免内存溢出
- 结果排序确保输出顺序一致

## 注意事项

1. 需要确保`pinyin.txt`文件存在于项目根目录
2. 程序运行时可能需要几秒到几十秒的时间，取决于计算机性能
3. 输出文件可能包含大量组合（通常是几千到几万个）
